## Processo

### Passo 1 — Entender o cenário
Se não fornecido, pergunte:
- "Qual é o produto ou serviço sendo vendido?"
- "Qual é o momento?" (primeiro contato / follow-up / reativação / contorno de objeção)
- "Qual é o canal?" (ligação, WhatsApp, e-mail, reunião)

### Passo 2 — Gerar o script

**Estrutura base para primeiro contato:**

1. **Abertura** — identificar-se, mencionar contexto de onde veio o lead
2. **Gancho** — motivo relevante para o contato (não "para apresentar")
3. **Pergunta de situação** — entender o momento do prospect
4. **Proposta de próximo passo** — agendar conversa, enviar material, marcar demo

**Estrutura para follow-up:**

1. Retomar o contexto da última conversa
2. Trazer algo novo de valor (dado, case, novidade)
3. Checar se houve mudança na situação
4. Propor próximo passo claro

**Estrutura para reativação de lead frio:**

1. Reconhecer o tempo sem contato sem se desculpar em excesso
2. Mencionar mudança relevante (novo produto, resultado de cliente, contexto de mercado)
3. Pergunta aberta sobre o momento atual
4. CTA simples e sem pressão

**Contorno de objeções comuns:**
- "Não tenho tempo agora" → [resposta]
- "Está caro" → [resposta]
- "Já tenho um fornecedor" → [resposta]
- "Preciso pensar" → [resposta]
- "Manda mais informações" → [resposta]

### Diretrizes
- Nunca soar robotizado — o script é guia, não decoreba
- Perguntas abertas > afirmações
- Sempre terminar com próximo passo concreto
- Tom consultivo, não de vendedor empurrando produto

### Passo 3
"Quer que eu escreva as respostas para as objeções específicas do seu produto?"
