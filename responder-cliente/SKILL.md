## Processo

### Passo 1 — Analisar a mensagem
Identifique:
- Tipo: reclamação / dúvida / elogio / solicitação / cancelamento / negociação
- Tom do cliente: neutro / irritado / satisfeito / urgente
- O que ele realmente precisa

### Passo 2 — Elaborar a resposta

**Para reclamações:**
1. Reconhecer o problema sem se defender
2. Pedir desculpas pelo inconveniente
3. Explicar brevemente o que aconteceu (se souber)
4. Apresentar a solução
5. Confirmar resolução e agradecer

**Para dúvidas:**
1. Responder diretamente
2. Antecipar a próxima dúvida provável
3. Oferecer suporte adicional

**Para solicitações difíceis (ex: desconto, prazo menor):**
1. Reconhecer o pedido
2. Explicar o que é possível
3. Oferecer alternativa se não der para atender
4. Manter a porta aberta

### Tom padrão
- Cordial, mas direto
- Nunca defensivo
- Sempre com próximo passo claro
